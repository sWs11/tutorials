# Удобная история буфера обмена

В Kubuntu из коробки есть утилита истории буфера обмена. Ей можно воспользоваться нажав на иконку в трее возле часов.

![Image alt](https://raw.githubusercontent.com/sWs11/tutorials/master/kubuntu_first_steps/images/3.1.png)

Выбрав одно из значений, его можно вставить в нужное место. Но не всегда удобно тянутся за мышкой что-то там выбирать, кликать. Гораздо проще когда на это все есть сочетания клавиш. И это можно сделать.

Нажав правой кнопкой мыши в трее на значек буфера обмена появиться контекстное меню размером в один елемент - "Настроить буфер обмена". Кликаем по нему.

Выбираем пункт **Комбинации клавиш**, затем кликаем на строке **Открывать Klipper на месте указателя мыши**, в колонке **Глобальная**, жмем на кнопку напротив пункта **Другая** и жмем удобно сочетания клавиш. Для меня это `meta + v` (`meta` - это тоже самое что и `windows`).

![Image alt](https://raw.githubusercontent.com/sWs11/tutorials/master/kubuntu_first_steps/images/3.2.png)

Применяем изменения. Нажимаем указаную комбинацию клавиш, и теперь история буфера обмена под курсором мыши. Выбрать что-то можно клавишами "Вверх", "Вниз", "Enter". 

![Image alt](https://raw.githubusercontent.com/sWs11/tutorials/master/kubuntu_first_steps/images/3.3.png)
